# 1. Load Instruction
    LUI → Load Upper Immediate (loads a 20-bit immediate into the upper bits of a register).

# 2. Immediate Instructions
    ADDI → Adds an immediate value to a register.
    ANDI → Logical AND operation between a register and an immediate.
    ORI → Logical OR operation between a register and an immediate.
    XORI → Logical XOR operation between a register and an immediate.

# 3. Arithmetic and Logical Instructions
    ADD → Adds a register to a register.
    SUB → Subtracts a register to a register.
    AND → Logical AND operation between registers.
    OR → Logical OR operation between registers.
    XOR → Logical XOR operation between registers.

# 4. Flow Control Instructions
    JAL → Jump and Link (jumps to an immediate address and stores the return).
    BEQ → Branch if two registers are equal.

# 5. Memory Instructions
    LW → Load Word (loads a 32-bit value from memory to a register). 
    SW → Store Word (stores a 32-bit value from memory into a register).
